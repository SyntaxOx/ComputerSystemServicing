FLOWCHARTING

TERMINOLOGIES:

ALGORITHM
- Finite sets of instructions that specify a sequence of steps to solve a problem.

FLOWCHART
- It use symbols to define the logical sequence of steps to solve a problem.

PROGRAM FLOWCHART
- Used to define the logical sequence of steps to be carried out preparing a computer program.

SEQUENCE STRUCTURE PROGRAM
- Process is executed in a straight forward manner.


FLOWCHART SYMBOLS:

TERMINAL
- Signify the beginning and end of flowchart.

PREPARATION / INITIALIZATION
- signifies the preparation of data or used to select initial conditions.

INPUT / OUTPUT
- Data are to be read into an input device or data are to be passed from memory to an output device.

PROCESSING
- Performs any calculation that are to be done.

DECISION
- Used when two alternative execution paths are possible.

ON PAGE CONNECTOR
- Shows entry or exit from one page to another page.

OFF PAGE CONNECTOR
- Designates entry or exit from one page to another page.

FLOWLINES
- signifies the process that is to be executed next.

PROGRAMMING LANGUAGE

CATEGORIES OF PROGRAMMING LANGUAGE

Machine or Low Level Language
- Machine language, also known as machine code, is the lowest level of programming language. It consists of binary instructions that can be directly executed by a computer's central processing unit (CPU). Each instruction is represented by a sequence of 0s and 1s, which correspond to specific operations and data manipulations performed by the computer's hardware.

Symbolic or Middle Level Language
- Permits the programmer to use convenient symbols or mnemonics (memory aids) when writing a program. Symbolic programming language refers to a type of programming language that uses symbols or expressions to represent computations, operations, and algorithms

High Level Language
- Statements closely resemble the English language statements. Languages have been designed to process problems of a scientific or engineering nature and some business problems application.


FEATURES OF PROGRAMMING LANGUAGE

BASIC
- Beginners All-Purpose Symbolic Instruction Codes

  • Created by John Kemeny and Thomas Kurtz
  • Easy to learn. Ideally suited for teaching student without any programming background.
  • Interactive language , one program is compiled by instructions as it is read into the computer and where the compiler has various editing features that allow the program to be modified as it is inputted.

COBOL
- (Common Business Oriented Language) a business language that can be run on any computer makes and models.

  • The language facilitates the used of structured programming techniques. Designed to standardized programming and make debugging easier.
  • Advantage of which, it employs the vocabulary of the business world and tend to be self documenting.
  • More difficult to learn than BASIC, but easier to use than machine oriented symbolic programming language.

FORTRAN
- (Formula Translator)

  • It is the oldest HLPL which was develop in 1957 by IBM to simplify statistical scientific and engineering calculations.
  • Has Features for determining Logarithms, trigonometric etc. Created for scientist, engineers and mathematicians.
  • Primary advantage is excellent mathematical capability and compactness.
  • Another its compiler needs relatively small amount of memory thus can be used in smaller computer

PASCAL
- named after Blaise Pascal, the mathematician. It was devised by Nicklause Wirth between 1968 and 1971.

  • The language facilitates the used of structured programming techniques. Designed to standardized programming and make debugging easier.
  • Pascal was adopted as the primary programming language for microcomputers.
  • One of the most important language in the 1980's

PL/1 (Programming Language 1)
- HLPL Designed to meet the needs of both scientific and commercial computer users.

  • With combined advantages of COBOL and FORTRAN for application in both scientific and commercial problems.
  • Disadvantage of which is used primarily for IBM computers only. It is not readily for other brands of computers.

RPG
- (Report Program Generator)

  • Symbolic Language ideally suited for creating printed reports from input media.
  • Ideally suited for mini-computer because it does not require large storage requirements.

APL
- (A Programming Language)

  • Designed by Kenneth E. Iverson at Harvard in the early 1960's
  • Designed to permit users to specify complex algorithm and logical expressions. Designed to solve immediate real world problems.

C Programming
- is a middle language since it combines elements of high-level language with the functionalism of assembly language.

  • A simple core language, such as math functions or file handling provided by a standard set of library routines.
  • Focus on procedural programming paradigm which facilitates programming on structured style.

JAVA
- Java is a powerful general-purpose programming language. It is used to develop desktop and mobile applications, big data processing, embedded systems, and so on. According to Oracle, the company that owns Java, Java runs on 3 billion devices worldwide, which makes Java one of the most popular programming languages.


Perl
- is a programming language developed by Larry Wall, especially designed for text processing. It stands for Practical Extraction and Report Language. It runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX.


PHP (recursive acronym for PHP: Hypertext Preprocessor )
- is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

  • PHP is a server side scripting language. that is used to develop Static websites or Dynamic websites or Web applications. PHP stands for Hypertext Pre-processor, that earlier stood for Personal Home Pages. PHP scripts can only be interpreted on a server that has PHP installed.
